# init-me

## 安装

```
npm install init-me -g
```

## usage

###

### seed 包安装

```
init install init-me-seed-initialize
```

### seed 包卸载

```
init uninstall init-me-seed-initialize
```

### seed 包列表

```
init list
```

### 运行

```
init
init path/to/dir
```

### 重置配置

```
init reset
```

### 显示工具路径

```
init --path
init -p
```

### 显示版本

```
init --version
init -v
```

### 显示帮助

```
init --help
init -h
```

### 安装本地 seed 包 (用于开发)

```
init link
```

### 卸载本地 seed 包 (用于开发)

```
init unlink
```

## seed 包开发

初始化

```
yarn create init-me-seed
```
